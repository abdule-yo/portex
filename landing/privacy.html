Privacy Policy

Effective: February 2026 · Last updated: February 2026

Summary

Portex is designed around one principle: your data stays on your device. We do not operate servers that receive, store, or process your conversations.

Data We Store

All data is stored exclusively in chrome.storage.local on your own machine:

  • Saved chat sessions (messages, titles, timestamps)
  • Generated summaries
  • Your API key (encrypted at rest by Chrome — never readable by us)
  • Extension settings (preferred model, theme)

Data We Never Collect

  • We have no backend server — there is nothing to send data to.
  • We do not use analytics, tracking pixels, or third-party SDKs.
  • We do not transmit your conversations to any service we operate.

Third-Party API Calls (Optional)

If you choose to enter an OpenAI or Anthropic API key, the extension will send your session content directly from your browser to those providers' APIs for summarization. This is optional — local compression works without any API key. Portex does not proxy, log, or intercept these requests.

Please review the privacy policies of those providers separately:

  • OpenAI Privacy Policy
  • Anthropic Privacy Policy

Permissions Used

  • storage — to save sessions locally in your browser.
  • tabs / activeTab / scripting — to read the current chat page and inject context into the target provider's input box. We only access tabs you explicitly interact with via the extension popup.

Changes

If this policy changes materially, we will update the "Last updated" date above and note the changes in the Chrome Web Store listing.
